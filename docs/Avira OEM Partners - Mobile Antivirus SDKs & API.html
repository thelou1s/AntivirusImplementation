    Mobile Antivirus API

    Best-in-class in both protection and usability

    IT security is only as strong as its weakest link. Bring your own device (BYOD) is a phenomenon that many companies try to stop because of the potential dangers that it brings.

    The rise of mobile devices with powerful processors and more RAM led to an increase in mobile malware. Use Avira’s Mobile Antivirus API (MAVAPI) in your products to identify malicious files right after download and malicious apps even before they’re fully installed on an Android mobile phone or tablet.

    MAVAPI, integrated into Avira’s award-winning Android security product, uncovers apps that misuse device data, monitor behavior, and send premium text messages.

    MAVAPI uses only an offline database with a very small footprint for signature updates. It is created to have a very low power consumption and is available on Android starting with version 2.2. It supports the mobile chipsets starting with ARM EABI 5.
